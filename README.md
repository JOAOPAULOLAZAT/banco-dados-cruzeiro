# Projeto SQL - Mini Mundo (ONG)

Este repositório contém alguns scripts SQL básicos feitos para testar
criação e manipulação de dados de um banco relacional baseado no mini-mundo da ONG.

# Arquivos do repositório

# 01_inserts.sql
Script com alguns INSERTs para colocar dados iniciais nas tabelas principais.

# 02_selects.sql
Contém algumas consultas SELECT usando WHERE, ORDER BY, LIMIT e JOIN.

# 03_updates_deletes.sql
Contém 3 comandos UPDATE e 3 DELETE para praticar alterações.

# Como executar
1. Crie o banco no MySQL Workbench ou PGAdmin.
2. Execute 01_inserts.sql.
3. Execute 02_selects.sql.
4. Execute 03_updates_deletes.sql.
